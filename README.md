# Scraperr
A web scraper to scrape political news(related to election) from various news websites

## Instructions:
Clone the repo:<br>
`git clone https://github.com/sinkobobux/Scraperr.git`

Install the dependencies with:<br>
`pip install -r requirements.txt`

Run with:<br>
`python main.py`

The default behavior of the program is to save the scraped data as individual text files.<br>
To save to a json file, use the **--json** argument<br>
`python main.py --json`
